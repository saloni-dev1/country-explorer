## Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/saloni-dev1/country-explorer.git
   cd country-explorer
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Run the development server:**
   ```sh
   npm run dev
   ```
4. **Open [http://localhost:3000](http://localhost:3000) in your browser.**
